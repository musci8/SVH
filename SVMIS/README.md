# SVMIS

This folder contains the code for extracting Statistically Validated Maximal Interacting Sets (SVMIS) and for analyzing empirical datasets.


## Setup

Install the necessary dependencies with:
```conda env create -f environment.yml```


## Usage

The folder `notebook/` containst the Jupyter Notebooks used to generate the figures in the paper.

## Data

Instructions to download the data is placed inside the folders in `data/`.
